Tokens are stored in this browser only.

Where do these come from? Airtable · Anthropic
Airtable — create a token at airtable.com/create/tokens with scopes data.records:read, data.records:write, schema.bases:read, granted to Fert & Foliar and Roseberry Planner.
Anthropic — only needed for the Seed book page, which reads a photo of the seeding book. Create a key at console.anthropic.com and set a low monthly spend cap. Sent only to Anthropic. Leave blank to skip.
TickTick optional — tick off tasks in the field
Register a free app at developer.ticktick.com with redirect URI https://tjdscgit.github.io/ticktick-oauth-redirect/, paste its Client ID/Secret above and save, then connect each person's own TickTick account below — this only needs doing once per person.
Google Calendar optional — push this week to a shared calendar
In Google Cloud Console, enable the Google Calendar API, configure the OAuth consent screen (External, scope .../auth/calendar, add the farm's Google account as a test user), then create an OAuth Web application Client ID with Authorized JavaScript origin https://tjdscgit.github.io. Paste the Client ID above and save, then connect — this only needs doing once. Events land on a dedicated "Roseberry Farm Tasks" calendar, not your primary one.
